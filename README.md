## About Me

I focus on turning operational requirements into maintainable software: secure admin flows, database-backed workflows, clean interfaces, API integrations, and documentation that helps teams keep systems running. My work spans government-service platforms, CMS and portfolio tools, reporting dashboards, Android apps, and AI-assisted prototypes.

Currently, I work with IT and consular-service systems at the **Bangladesh High Commission in Brunei**. Previously, I contributed to digital-service initiatives at **a2i, ICT Division, Bangladesh**, along with software engineering roles across web and platform teams.

- **Front-end applications:** Vue, React, Bootstrap, Tailwind, responsive dashboards, and UI-to-template implementation
- **Mobile applications:** Android clients, API-based workflows, activities/fragments, and service-focused app modules
- **AI and computer vision:** chatbot prototypes, YOLO/OpenCV experiments, face recognition, and AI-assisted decision support
- **Technical enablement:** documentation, architecture notes, deployment support, and cross-team collaboration
